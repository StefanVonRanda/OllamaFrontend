# OllamaFrontend
A dead simple frontend for Ollama served with `Bun`

![Screenshot](Screenshot.webp)


#### Requirements

- Bun
- Ollama
- At least 1 local model 



To install dependencies:

```bash
bun i
```

To run:

```bash
bun run start
```
